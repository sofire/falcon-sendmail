# 简介

为open-falcon提供短信、邮件的发送服务

参考 mail-provider & sms-provider 接口文档
https://book.open-falcon.org/zh/quick_install/judge_components.html



# 安装

依赖：
- php 5.5
- php composer


```
composer update 
```



# 配置

## Mail Provider
使用smtp发送邮件
